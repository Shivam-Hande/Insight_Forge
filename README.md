## Insight Forge – AI Customer Analytics Platform

Insight Forge is a machine learning powered analytics platform that predicts:

• Customer Lifetime Value (CLV)  
• Customer Churn Risk  

## Features
- CLV prediction model
- Customer churn prediction
- Interactive analytics dashboard
- Flask-based web application

## Tech Stack
Python, Flask, Pandas, Scikit-learn, HTML, CSS, JavaScript
